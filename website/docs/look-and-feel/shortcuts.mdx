# Keyboard Shortcuts

<p className="header-description">Navigate Sparky quickly with keyboard shortcuts.</p>

All shortcuts use **⌘** on macOS and **Ctrl** on Windows.

## Chat

| Action | macOS | Windows |
|--------|-------|---------|
| New chat | ⌘ N | Ctrl+N |
| Search chats | ⌘ K | Ctrl+K |
| Delete chat | ⌘ ⌫ | Ctrl+Backspace |
| Print chat | ⌘ P | Ctrl+P |
| Focus input | ⌘ L | Ctrl+L |
| New line | ⇧ ↩ | Shift+Enter |

## Navigation

| Action | macOS | Windows |
|--------|-------|---------|
| Chats | ⌘ 1 | Ctrl+1 |
| Knowledge | ⌘ 2 | Ctrl+2 |
| Connections | ⌘ 3 | Ctrl+3 |
| Settings | ⌘ 4 | Ctrl+4 |
| Toggle sidebar | ⌘ B | Ctrl+B |

:::tip
You can also view all shortcuts in **Settings → Look & Feel**.
:::
