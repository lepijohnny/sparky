# Flag, Archive and Label

<p className="header-description">Mark important conversations by flagging or archive them once they are no longer needed. Use labeling for additional organization.</p>

## Flagging

Flag important conversations to mark them for follow-up. Flagged chats appear with a visual indicator in the sidebar and can be filtered quickly.

## Archiving

Archive conversations you no longer need in your active list. Archived chats are hidden from the main view but remain fully searchable and accessible. Nothing is deleted — archive is reversible at any time.

## Labeling

Create custom labels to organize your conversations. Apply multiple labels to a single chat for flexible categorization. Filter by label to quickly find related conversations across different topics or projects. Labels appear in the chat list but also above the chat input field.

Each chat can be quickly labeled or unlabeled by typing `#` in the input field, which triggers the label context menu.
