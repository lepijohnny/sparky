# What's Coming Next

<p className="header-description">Planned features and improvements.</p>

## v0.3.0

- [x] macOS and Windows support
- [x] Multi-provider LLM (Claude, Copilot, Gemini, Mistral, Ollama)
- [x] Local RAG with hybrid search and reranking
- [x] Service connections (REST + MCP)
- [x] Chat organization (flags, labels, archive, search)
- [x] Meta-chat (control the app through conversation)
- [x] Print and export conversations
- [x] Custom themes
- [x] Auto updater
- [x] Application signing

## v0.4.0

- [x] Skills system
- [x] Bash and Python code execution (unsecured, opt-in)
- [x] Shortcut keys (macOS and Windows)

## v0.5.0

- [X] Tools activity improvements (clarify tool usage in the conversation view)
- [x] OpenAI provider support
- [x] Mistral provider support
- [x] Markdown lexer and renderer improvements (diff view, lang highlighting)

## v0.6.0

- [x] Linux support (descoped to future relases due to WebKitGTK limitations on Linux)
- [x] Universal file extraction via [markit](https://github.com/Michaelliv/markit) — PDF, DOCX, PPTX, XLSX, HTML, EPUB, CSV, images, audio, URLs, and more
- [ ] MacOs intel support

## v0.7.0

- [ ] Drop custom extractor plugin system in favor of built-in markit support
- [ ] Automation triggers (cron)

## v0.8.0

- [ ] Better attachment support, more formats, drag-and-drop
- [ ] Conversation branching
- [ ] Core browser support (navigation, context extraction)

## v0.9.0

- [ ] Extended browser support (form filling, login)
- [ ] Sandboxed code execution

## Beyond

- [ ] CLI companion tool
- [ ] Skills hub
- [ ] Sharable conversation links
- [ ] Improved permission system
