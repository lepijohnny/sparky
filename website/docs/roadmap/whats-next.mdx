# What's Coming Next

<p className="header-description">Planned features and improvements.</p>

## v0.3.0 — Current Release

- [x] macOS and Windows support
- [x] Multi-provider LLM (Claude, Copilot, Gemini, Ollama)
- [x] Local RAG with hybrid search and reranking
- [x] Service connections (REST + MCP)
- [x] Chat organization (flags, labels, archive, search)
- [x] Meta-chat (control the app through conversation)
- [x] Print and export conversations
- [x] Custom themes
- [x] Auto updater
- [x] Application signing

## v0.4.0

- [x] Skills system
- [x] Bash and Python code execution (unsecured, opt-in)
- [x] Shortcut keys (macOS and Windows)

## v0.5.0

- [ ] OpenAI provider support
- [ ] Mistral provider support
- [ ] Markdown lexer and renderer improvements (diff view, lang highlighting)

## v0.6.0

- [ ] Linux support

## v0.7.0

- [ ] More extractor plugins
- [ ] Better attachment support, more formats
- [ ] Drag and drop file attachments
- [ ] Conversation branching
- [ ] Core browser support (navigation, context extraction)

## v0.8.0

- [ ] Extended browser support (form filling, login)
- [ ] Sandboxed code execution

## Beyond

- [ ] CLI companion tool
- [ ] Skills hub
- [ ] Sharable conversation links
- [ ] Improved permission system
