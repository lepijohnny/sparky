# What's Coming Next

<p className="header-description">Planned features and improvements.</p>

## v0.3.0 — Current Release

- macOS and Windows support
- Multi-provider LLM (Claude, Copilot, Gemini, Ollama)
- Local RAG with hybrid search and reranking
- Service connections (REST + MCP)
- Chat organization (flags, labels, archive, search)
- Meta-chat (control the app through conversation)
- Print and export conversations
- Custom themes
- Auto updater

## v0.4.0

- Application signing
- Drag and drop file attachments
- Skills system
- Bash and Python code execution (unsecured, opt-in)
- Markdown lexer and renderer improvements
- Shortcut keys (macos and windows)

## v0.5.0

- Linux support
- Improved onboarding flow
- More extractor plugins
- Better attachment support more formats
- Conversation branching

## Beyond

- Sandboxed code execution
- CLI companion tool
- Skills hub?
- Sharable conversation links
