# What's Coming Next

<p className="header-description">Planned features and improvements.</p>

## v0.3.0

- [x] macOS and Windows support
- [x] Multi-provider LLM (Claude, Copilot, Gemini, Mistral, Ollama)
- [x] Local RAG with hybrid search and reranking
- [x] Service connections (REST + MCP)
- [x] Chat organization (flags, labels, archive, search)
- [x] Meta-chat (control the app through conversation)
- [x] Print and export conversations
- [x] Custom themes
- [x] Auto updater
- [x] Application signing

## v0.4.0

- [x] Skills system
- [x] Bash and Python code execution (unsecured, opt-in)
- [x] Shortcut keys (macOS and Windows)

## v0.5.0

- [X] Tools activity improvements (clarify tool usage in the conversation view)
- [x] OpenAI provider support
- [x] Mistral provider support
- [x] Markdown lexer and renderer improvements (diff view, lang highlighting)

## v0.6.0

- [x] Linux support (descoped to future relases due to WebKitGTK limitations on Linux)
- [x] Universal file extraction via [markit](https://github.com/Michaelliv/markit) — PDF, DOCX, PPTX, XLSX, HTML, EPUB, CSV, images, audio, URLs, and more
- [x] MacOs intel support (descoped to future releases due to low demand and development complexity)
- [x] Drop custom extractor plugin system in favor of built-in markit support
- [x] Attachments drag-and-drop support
- [x] 1M context window support
- [x] Tool management improvements (large tool output cached in files)
- [x] Chat permission system (Approve Once, Approve All, Deny)

## v0.7.0

- [x] Routines (cron)
- [x] Assistant labels for permission, connection, skill and routine
- [x] Conversation steering messages (inject messages into the conversation while the agent is working)
- [x] Conversation branch/copy/edit/delete (start new chat from specific message with the previous context)

## v0.8.0

- [ ] DeepSeek API support (Anthropic OAuth Pro/Max discontinued since 04.04.2026)
- [ ] Events and triggers
- [ ] Permission system improvements (per-chat trust scopes, more granular permissions, less intrusive, more controllable)
- [ ] Core browser support (navigation, context extraction)

## v0.9.0

- [ ] Extended browser support (form filling, login)
- [ ] Sandboxed code execution? (macos)

## Beyond

- [ ] CLI companion tool
- [ ] Skills hub
- [ ] Sharable conversation links
